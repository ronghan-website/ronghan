# Dr. Rong Han - Mathematics Researcher Website

A professional, modern personal website for a mathematics researcher, built with clean HTML, CSS, and JavaScript.

## Features

✨ **Professional Design**
- Clean, modern aesthetic suitable for academic professionals
- Gradient hero section with prominent call-to-action buttons
- Responsive design that works on all devices

📚 **Content Sections**
- **About**: Professional bio and areas of expertise
- **Research**: Key research interests and specializations
- **Publications**: Showcase of academic publications
- **Teaching**: Courses and educational contributions
- **Contact**: Multiple ways to get in touch

🎨 **Design Features**
- Sticky navigation bar
- Smooth scrolling between sections
- Hover effects and animations
- Professional color scheme (blue and amber)
- Font Awesome icons for visual elements

📱 **Responsive**
- Mobile-friendly design
- Adapts to tablets, desktops, and large screens
- Touch-friendly navigation

## Files

- `index.html` - Main website structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations
- `README.md` - This file

## Customization

### Personal Information
Edit the following in `index.html`:
- Name and title in the hero section
- Bio in the About section
- Research interests and areas
- Publications details
- Teaching courses
- Contact information

### Styling
- Colors can be customized in the CSS variables at the top of `styles.css`
- Font family, spacing, and other design elements can be adjusted

### Adding Content
- Add new publications to the publications list
- Update research interests
- Add new courses or teaching materials
- Include links to your Google Scholar, ResearchGate, GitHub, etc.

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript** - Interactive features
- **Font Awesome 6** - Icons

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content with your information
4. Deploy to GitHub Pages or your preferred hosting

## Deployment with GitHub Pages

1. Go to your repository Settings
2. Navigate to Pages section
3. Select `main` branch as the source
4. Your site will be available at `https://ronghan-website.github.io/ronghan`
